Insert a gallery based on an rss feed
-------------------------------------

Makes it easy to add a photo gallery based on an RSS feed from sites such as Flickr or Photobucket.

To add a gallery to a page:
    Edit the page and add a piece of text like this:
    [[galleryrss:rss url]]
    eg [[galleryrss:http://picasaweb.google.co.uk/data/feed/base/user/Mike.from.MikesFlat/albumid/5062149466400611009?kind=photo&hl=en_US]]

Tested with feeds from:

Picasa Web Albums
    eg [[galleryrss:http://picasaweb.google.co.uk/data/feed/base/user/Mike.from.MikesFlat/albumid/5062149466400611009?kind=photo&alt=rss&hl=en_US]]
Flickr
    eg [[galleryrss:http://api.flickr.com/services/feeds/photos_public.gne?id=38068860@N00&amp;lang=en-us&format=atom]]
Photobucket
    eg [[galleryrss:http://feed.photobucket.com/images/auckland/feed.rss]]
